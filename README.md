# Caesar-Cipher_tool

A simple cryptography tool written in Python that can encrypt, decrypt,
and brute-force Caesar cipher messages.

## Features
- Encrypt text using Caesar Cipher
- Decrypt encrypted messages
- Brute-force attack to recover plaintext

## Technologies Used
- Python 3

## Example

Input:
HELLO

Shift: 3

Output:
KHOOR

## Project Structure

caesar_cipher.py -> encryption & decryption logic
